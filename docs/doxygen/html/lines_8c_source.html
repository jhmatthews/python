608 //OLD state atoms which correspond to scattered photons, i.e. the portion which are
609 //OLD excited by radiation and return to the ground state via spontaneous emission.
610 //OLD
611 //OLD Description:
612 //OLD
613 //OLD In the radiative transfer equation for lines, we separate the fraction of photons
614 //OLD which are "absorbed" and the fraction which are "scattered".
615 //OLD
616 //OLD If line_mode==0, the atomosphere is a completely absorbing and no photons
617 //OLD will be scattered. In this mode, assuming the wind is a source
618 //OLD of emission, the emissivity will be the Einstein A coefficient
619 //OLD If line_mode==1, the atmosphere is completely scattering, there will be no
620 //OLD interchange of energy between the photons and the electrons
621 //OLD as a result of radiation transfer
622 //OLD If line_mode==2, then a simple single scattering approximation is applied in which
623 //OLD case the scattered flux is just A21/(C21+A21*(1-exp(-h nu/k T_e).
624 //OLD If line_mode==3, then radiation trapping is included as well. The basic idea
625 //OLD is to calculate the average number of scatters in a single
626 //OLD interaction and there is heat lost in each of these scatters.
627 //OLD Notes:
628 //OLD It may be more efficient to combine several of these routines in view
629 //OLD of the fact that the exp is calculated several times
630 //OLD

634 //OLD two-level atom population. Also rewrote the

684 scattering_fraction (line_ptr, xplasma)
685  struct lines *line_ptr;
686  PlasmaPtr xplasma;
687 {
688  double q, escape;
689  double a, c, z;

693 
694  if (geo.line_mode == 0)
695  return (0.0); //purely absorbing atmosphere
696 
697  else if (geo.line_mode == 1)
698  return (1.); //purely scattering atmosphere
699 
700  //Populate variable from previous calling structure
701  ne = xplasma->ne;

704
