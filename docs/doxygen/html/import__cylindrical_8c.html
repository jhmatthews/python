int import_cylindrical ( int  ndom,
char *  filename 
)

Read the an arbitray wind model in cylindrical coordinates.

Parameters
[in]ndomThe domain number for the imported model
[in]filenameThe file containing the model to import
Returns
Always returns 0

This routine just reads in the data and stores it in arrays

Notes

The basic data we need to read in are

  • i, j, r z v_x v_y v_z rho (and optionally T)

where v_x,_v_y,v_z are the velocities in the x,z plane and where

  • i is the column number (Thus i corresponds to ndim)
  • j is the row number (and z coresponds to mdim)

We assume that all of the variables are centered, that is we are not assuming that we are giving rho at the center of a cell, but that r and v_r are at the edges of a cell. This is someghing that would presumable be easy to change

Note that we assume that the data are being read in in the same order as printed out by windsave2table, that is that the first "column" is read in, and then the second "column".

double rho_cylindrical ( int  ndom,
double *  x 
)

Get the density for an imported cylindrical model at x.

Parameters
[in]ndomThe domain for the imported model
[in]xA position
Returns
The density in cgs units is returned

This routine finds rho from the imported model at a position x. The routine does not interpolate rho, but simply locates the cell associated with x

Notes

This routine is really only used to intialize rho in the Plasma structure. In reality, once the Plasma structure is initialized we always interpolate within the plasma structure and do not access the original data.
