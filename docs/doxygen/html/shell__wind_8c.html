Detailed Description

Routines needed for a single shell wind model.

Author
ksl,nsh
Date
May, 2018

These routines were developed (by NSH) primarily for diagnostic purposes, e.g for calculating ionization models for comparison with Cloudy. The routines are in many ways similar to stellar wind model routines, but after a discussion in 2015 (See issue #172), we decided to keep it.

Parameters

Returns

This routine gets the inputs needed for the shell_wind model and translates them into those needed for the equvalent stellar wind model. Several arrays specific to the shell_wind model are also intialized.
