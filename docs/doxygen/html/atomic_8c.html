Detailed Description

Routines to calculate rates and x-sections making use of the atomic data.

Author
ksl
Date
March, 2018

Notes

These routines are intended to be Python-independent so that the can be used with other programs. As a result python.h should not be included.

Note that other routines should be added here to make the atomic data routines more indepenent of Python.

[in,out]doublefreq frequency at which the x-section should be calculated

sigma_phot uses Verner et al.'s interpolation formulae for the photoionization crossection to calculate the bound free (or photoionization) optical depth.

Notes
