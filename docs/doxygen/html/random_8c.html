Create a photon direction "lmn" in the hemisphere with the vector "north pointin go the north pole according to the Eddingon approximation

[in]doublenorth [] The direction of local north

Create a photon direction "lmn" in the hemisphere with the vector "north" pointing to the "north pole" pole of the hemispere in the case where the photon is originating in a photosphere. Another way of saying this is that north is the normal to surface at the point at which the photon originates.

The photon directions will be distributed according to the Eddington approximation

Notes

The routine calls vcos which actually containes the Eddinggton approximation (aka linear limb darkening)

Jumps were added to include more points near 90 degrees.
